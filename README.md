# 드워프 포트리스 DFI18n 국제화 모드 데이터 - 한국어

이 저장소는 드워프 포트리스 DFI18n 국제화 모드의 한국어 번역 데이터입니다. 설치하려면 《드워프 포트리스》 Steam 창작마당에서 구독하세요.

## 저작권 및 라이선스 정보

한국어 번역 데이터 생성 작업 과정에서 참조 및 사용된 중국어판 로고, 중국어판 간단 사전, 중국어판 규칙 세트 데이터의 저작권은 「[드워프 포트리스 중국어 위키](https://dfzh.huijiwiki.com/) 번역팀」 번역자들에게 있으며, 라이선스 및 사용 허가 정보는 [라이선스(LICENSE.md)](LICENSE.md) 문서를 참고하세요.

번역 데이터에 포함된 폰트는 SIL Open Font License로 배포되며, 자세한 내용은 [dfi18n-data/fonts/ko/LICENSE.txt](dfi18n-data/fonts/ko/LICENSE.txt)를 참고하세요.
